barak.absorb.T_to_b¶

barak.absorb.T_to_b(atom, T)[source]¶

Convert temperatues in K to b parameters (km/s) for an atom with mass amu.

Parameters :

atom : str or float

Either an abbreviation for an element name (for example ‘Mg’), or a mass in amu.

T : array_like

One or more temperatures in Kelvin.
Returns :

b : ndarray or float

The b value in km/s corresponding to each input temperature.
