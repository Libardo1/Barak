barak.sed.SED.calc_colour¶

SED.calc_colour(band1, band2, system='Vega')[source]¶

Calculates the colour band1 - band2.

system is either ‘Vega’ or ‘AB’.

mag_sigma : float
Add a gaussian random deviate to each magnitude, with sigma given by this value.
