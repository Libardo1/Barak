barak.absorb.b_to_T¶

barak.absorb.b_to_T(atom, bvals)[source]¶

Convert b parameters (km/s) to a temperature in K for an atom with mass amu.

Parameters :

atom : str or float

Either an abbreviation for an element name (for example ‘Mg’), or a mass in amu.

bvals : array_like

One or more b values in km/s
Returns :

T : ndarray or float

The temperature corresponding to each value in `bvals’.
