barak.pyvpfit.readatom¶

barak.pyvpfit.readatom(filename=None, debug=False, flat=False, molecules=False, isotopes=False)[source]¶

Reads in atomic transitions from a vpfit atom.dat file. Single argument is atom.dat filename.

Returns a dictionary of atoms.

if flat, return a flat array of atoms as well as a dictionary.

Examples

>>> at = readatom('data/atom.dat.gz')
