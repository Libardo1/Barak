barak.cosmology.core.Cosmology.comoving_transverse_distance¶

Cosmology.comoving_transverse_distance(z)[source]¶

Comoving transverse distance in Mpc at a given redshift.

This value is the transverse comoving distance at redshift z corresponding to an angular separation of 1 radian.

This is the same as the comoving distance if omega_k is zero (as in the current concordance lambda CDM model).
