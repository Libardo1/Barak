barak.sed.qso_template_sdss¶

barak.sed.qso_template_sdss(wa, z)[source]¶

Return a composite visible QSO spectrum at redshift z.

The SDSS composite spectrum as a function of F_lambda is returned at each wavelength of wa. wa must be in angstroms.

Only good between 700 and 8000 (rest frame).
