barak.pyvpfit.readf26¶

barak.pyvpfit.readf26(fh, res=None)[source]¶

Reads a f26 style file and returns a VpfitModel object. If the keyword res is given, this string provides the resolution information for the spectra fitted.

For example: res=’vsig=69.0’
