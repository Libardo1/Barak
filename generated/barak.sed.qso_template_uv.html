barak.sed.qso_template_uv¶

barak.sed.qso_template_uv(wa, z)[source]¶

Return a composite UV QSO spectrum at redshift z.

Wavelengths must be in Angstroms.

This is a smoothed version of the HST/COS EUV+FUV AGN composite spectrum shown in Figure 5 from Shull, Stevans, and Danforth 2012.

Only good between 550 and 1730 Angstroms (rest frame)
